You are not designing authentication screens.

You are designing an emotional arrival experience.

Your task is to create a SINGLE STANDALONE HTML FILE for the complete authentication experience of EZEE Prints.

Do not think in terms of Login Page, Signup Page, OTP Page, Forgot Password Page.

Think in terms of:

"Returning to Ezi's Study Nook."

This should feel like entering a peaceful room after a long day.

The user is not logging into software.

The user is coming home.
